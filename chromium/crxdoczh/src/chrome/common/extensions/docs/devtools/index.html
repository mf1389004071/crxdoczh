概述

Chrome 开发者工具（简称开发者工具）是 Google Chrome 浏览器中内建的网页编写和调试工具，开发者工具允许网页开发者访问浏览器以及网上应用的内部细节。开发者工具概述指出了最常用而且最有用的功能，如果您以前没有用过开发者工具，请从这里开始。即使您是一个经验丰富的网页开发者，您可能也会学会一些技巧。

    开发者工具窗口

    开发者工具按照任务分类来组织，窗口顶部的工具栏包含所有分类。工具栏中的每一项以及对应的面板可以使您关注某种类型的网页或应用信息，包括 DOM 元素、资源和源代码。

    总体说来，开发者工具包括八类主要工具：Elements（元素）、Resources（资源）、Network（网络）、Sources（源代码）、Timeline（时间线）、Profiles（性能分析）、Audits（审计）和 Console（控制台）。您可以使用 Ctrl+[ and Ctrl+] 快捷键在面板间移动。

    审查 DOM 和样式

    Elements（元素）面板允许您查看整个 DOM 树，还能审查和实时编辑 DOM 元素。当您需要寻找网页中某些部分的 HTML 片段时，您会经常访问 Elements（元素）标签页。例如，您可能想知道某个图片有没有 HTML id 属性及其属性值是什么。

    阅读更多关于审查 DOM 和样式的内容

    使用控制台

    JavaScript 控制台为测试网页和应用的开发者提供了两项主要功能：

    • 使用控制台 API 提供的方法，例如 console.log() 或 console.profile() 记录诊断信息。
    • 命令提示符，在其中您可以键入命令与文档和 Chrome 开发者工具交互。您可以直接在控制台中为表达式求值，还可以使用命令行 API 提供的方法，例如用 $() 选择元素或 profile() 开始 CPU 性能分析。

    阅读更多关于控制台使用的内容

    JavaScript 调试

    随着 JavaScript 应用程序复杂性的增加，开发者需要强大的调试工具迅速发现问题的原因并有效解决。Chrome 开发者工具包含许多有用的工具，使您更轻松地调试 JavaScript。

    审计

    Audits（审计）面板可以在网页加载时进行分析，并提供建议和优化方式，以便减少网页加载时间，提高用户察觉到（以及实际上）的响应速度。如果您希望更进一步的分析，我们还建议您安装 PageSpeed 扩展程序。

    提高渲染性能

    Timeline（时间线）面板为您提供加载和使用您的网上应用或网页时如何花费时间的完整概述，所有事件，从加载资源到分析 JavaScript、计算样式、重绘都会在时间线上表示出来。

    JavaScript 和 CSS 性能

    Profiles（性能分析）面板允许您分析网上应用或网页的执行时间和内存使用。Profiles（性能分析）面板包含几个性能分析工具：CPU 性能分析器、JavaScript 性能分析器 和 堆性能分析器。这些工具会帮助您理解资源在什么地方使用，便于您优化自己的代码：

    • CPU 性能分析器显示您的网页中哪些 JavaScript 函数花费时间执行。
    • 堆性能分析器显示您的网页中 JavaScript 对象和相关 DOM 节点的内存分布。
    • JavaScript 性能分析显示执行时间如何在您的脚本中花费。
