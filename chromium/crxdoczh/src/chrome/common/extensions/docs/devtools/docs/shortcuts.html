快捷键

开发者工具有一些内建的快捷键，开发者利用它们可以在日常工作中节约很多时间。下面列出了所有快捷键以及 Windows/Linux 和 Mac 平台上对应的按键。某些快捷键在整个开发者工具中都可以使用，而另外一些仅在某一个面板中有效，这些快捷键按照可以使用的位置分别列出。

打开开发者工具

要访问开发者工具，在 Google Chrome 浏览器中的任意网页或应用上，您可以使用以下任何一种方式：

  • 打开浏览器窗口右上角的 Chrome 菜单 ，并选择工具 > 开发者工具。
  • 右键单击网页中的任何元素并选择审查元素。
    • Windows / Linux Mac
    打开开发者工具 F12, Ctrl + Shift + I Cmd + Opt + I
    打开/切换审查元素模式和浏览器窗口 Ctrl + Shift + C Cmd + Shift + C
    打开开发者工具并激活控制台 Ctrl + Shift + J Cmd + Opt + J
    审查开发者工具（首先取消固定再按下快捷键） Ctrl + Shift + J Cmd + Opt + J

    如果要打开常用设置对话框请在开发者工具窗口打开时按下 ? 或 F1，按下 Esc 关闭设置对话框。
