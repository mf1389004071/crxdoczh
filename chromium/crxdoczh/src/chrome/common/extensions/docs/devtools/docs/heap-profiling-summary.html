收集分散的对象

该网页演示了如何使用堆性能分析器评估同一种对象占用空间的概况。

如下是用于参考的脚本源代码：

function Item(x)
{
  this.x = x;
}

function numbers()
{
  var result = new Array(10000);
  for (var i = 0, l = result.length; i < l; ++i)
    result[i] = new Item(i);
  return new Item(result);
}

function strings()
{
  var result = new Array(10000);
  for (var i = 0, l = result.length; i < l; ++i)
    result[i] = new Item(i.toString());
  return new Item(result);
}

function init()
{
  numberCache = numbers();
  stringCache = strings();
  documentCache = new Item(document.body.textContent.toLowerCase());
}

试一试：

  • 抓取堆快照
  • 打开 Summary（概述）视图

同一种类型的对象可能会分散在应用程序数据中，并且不经意间形成了很大一块浪费内存。Summary（概述）视图可以帮您发现过多的实例化。

例如，Item 对象在应用程序代码中用于不同的存储目的，所有对象都在 Summary（概述）视图中分类显示，无论它们的来源是什么，另外还提供了统计信息。
