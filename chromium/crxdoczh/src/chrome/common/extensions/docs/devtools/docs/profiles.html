Profiles（性能分析）面板

Profiles（性能分析）面板允许您分析网上应用或网页的执行时间和内存使用。Profiles（性能分析）面板包含两种性能分析器：CPU profiler（CPU 性能分析器）和 Heap profiler（堆性能分析器）。这些工具可以帮助您理解资源是如何消耗的，有助于您优化代码：

  • CPU profiler（CPU 性能分析器）显示执行时间如何在网页的 JavaScript 函数中花费。
  • Heap profiler（堆性能分析器）显示网页中 JavaScript 对象和相关 DOM 节点的内存分布。
  • CSS Selector Profiler（CSS 选择器性能分析器）记录 DOM 树中元素匹配选择器花费的总时间，时间线事件中的“Style Recalculation”（重新计算样式）也会捕获这一时间。
