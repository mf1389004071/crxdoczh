工作空间——保留开发者工具中编写的内容

简介

Chrome 开发者工具允许您更改网页中的元素或 CSS，并且能立即看到您的更改。但是如果您每次都需要将这些更改复制、粘贴到外部编辑器的话，这样对您的编写工作流程还有什么好处呢？工作空间可以帮您将这些更改保存在磁盘上，而不需要离开 Chrome 开发者工具进行这些操作。

使用工作空间，您可以在源代码面板中编辑任意类型的源文件，并将更改保存在磁盘中。您还可以将本地网页服务器上的资源映射到磁盘文件上，这样的话，当您更改和保存这些文件时，您可以像服务器托管一样查看它们。如果您设置了正确的映射关系，元素面板中的样式更改也可以自动保存在磁盘上。

将您的项目引入工作空间

为了让本地文件夹中的源文件能够在源代码面板中编辑，右键单击源代码左侧面板，选择 Add Folder to Workspace（在工作空间中添加文件夹）。这样操作之后会打开一个文件对话框，您可以选择要添加到工作空间中的新文件夹。（目前高亮的文件夹不会添加到工作空间中。）

Chrome 浏览器顶部显示黄色信息栏“DevTools 请求对[文件夹路径]的完整访问权限”时，单击允许按钮。

现在您就可以在 Chrome 浏览器中编辑该文件夹及其子文件夹中的任意源文件，在这种情况下“源文件”就不仅限于 HTML、CSS 和 JavaScript 了，可以是任何类型，包括 Markdown 和 JSON。

映射网络资源

工作空间的真正优势在于它能够将本地文件映射到 URL（或“网络资源”）。每当 Chrome 浏览器加载映射后的 URL 时，它会显示工作空间文件夹内容，代替网络文件夹内容，就像在网络上托管一样，但是您可以使用开发者工具修改它，并将更改保存到本地文件。

如果要将您的网站映射到本地工作空间文件夹：

  1. 在源代码面板中，右键单击或 Control+单击（Mac）网站中的文件。
  2. 选择 Map to File System Resource（映射至文件系统资源）。
  3. 从列表中选择对应文件（您可能需要键入文件名或一部分搜索字符串才能找到您需要的文件）。
  4. 重新加载 Chrome 浏览器中的网页。

现在源代码面板应该显示网站本地工作空间文件夹中的内容，而不是 localhost 源代码。

您还可以采用另一种方式实现这一点，使用 Map to Network Resource（映射至网络资源）将工作空间文件夹映射至 URL。请牢记这一点，不是所有网络资源文件都能在浏览器中加载，并映射到本地文件，但是所有本地文件都能映射至 URL。映射工作空间中的一个文件应该能将网站的大部分内容映射到相应的工作空间。

注意事项

工作空间可以大大简化您的工作流程，而不用在 Chrome 浏览器和外部编辑器之间来回切换。当然，还有一些事项要注意：

  • 元素面板中只有样式更改才会保留，DOM 更改不会保留。
  • 只要您将 CSS 文件映射至本地副本，元素面板中的样式更改就会立即保存（也就是说，元素面板中的更改会自动保存到文件中，而不用您显式保存）。
  • 如果您映射远程服务器而不是本地服务器上的文件，当您刷新网页时，Chrome 浏览器会再次从远程服务器重新加载网页。您的更改仍然会保存到磁盘，如果您继续在工作空间中编辑的话更改也会重新应用。

工作空间中的文件管理

除了编辑现有的文件，您还可以在工作空间中的本地目录内添加和删除文件。

添加文件

右键单击左侧源代码面板中的文件夹，并选择 New File（新建文件）。

删除文件

右键单击左侧源代码面板中的文件，并选择 Delete File（删除文件）。

您还可以选择 Duplicate File（复制文件）来复制某个文件，新文件会显示在源代码面板中，您可以输入新的文件名（不只是创建“Copy of myfile.txt”）。

刷新

您不仅可以直接在工作空间中创建（或删除）文件，源代码目录还会自动刷新并显示这些新文件。如果没有刷新，您随时都可以右键单击某个文件夹，并从弹出菜单中选择 Refresh（刷新）强制刷新。

如果您恰好更改了另一个编辑器中打开的文件，希望更改显示在开发者工具中的话，这一点也很有用。通常开发者工具能自动刷新，即使是在外部编辑器中编辑，但是如果您要重新编译 HTML 文件或 CSS 文件的话，您可能需要手动刷新。

搜索文件和在所有文件中搜索

搜索开发者工具中以通常方式加载的文件时，您可以按下 Ctrl + O（或 Mac 上的 Cmd + O）打开对话框并搜索文件名。在工作空间中您也可以这么做，只是它既搜索远程加载的文件，也会搜索工作空间文件夹中的本地文件。

还有另一种搜索文件内容的方式，所以您可以在工作空间中的所有文件以及开发者工具中加载的所有文件中搜索字符串。您既可以搜索字符串，也可以使用正则表达式，我们会匹配每一个文件或网页中出现的内容。

如果要在工作空间中的多个文件内搜索文本：

  1. 按下 Esc 键打开控制台抽屉，并单击 Console（控制台）旁边的 Search（搜索）标签页，打开搜索窗口。

    或

    按下 Ctrl + Shift + F（或 Mac 上的 Cmd + Opt + F）打开搜索窗口。
  2. 在搜索源代码框中输入查询内容，并按下回车键。如果您的查询内容是正则表达式或者需要区分大小写，请单击相应的复选框。
